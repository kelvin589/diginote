TokenUpdaterService

TokenUpdaterService class Null safety

A service which listens to user changes in FirebaseAuth to update the user's list of FCMTokens.

These tokens could be used, for example, to message every device linked to this user. The init method must be called to initialise TokenUpdaterService.
